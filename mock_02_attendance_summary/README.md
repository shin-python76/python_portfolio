# 📘 模擬案件②：勤怠データを読み込み、勤務日数・残業時間・有給日数を自動集計

## 🧾 クライアント要件書
社員ごとの勤怠データ（Excel）から、  
**勤務日数・残業時間・有給日数** を自動で集計し、月報を作成してください。

---

## 🔧 詳細要件
- **入力ファイル**：`attendance_2024_10.xlsx`  
- **列構成**：`社員名`, `日付`, `勤務時間`, `休暇区分`  
- **出力内容**：  
  - 集計結果をまとめた Excelファイル（`attendance_summary.xlsx`）  
  - 各社員ごとに「勤務日数」「残業時間」「有給日数」を算出  

---

### ⚙️ 処理内容
1. 勤怠データを pandas で読み込み  
2. 勤務日数＝勤務時間が0より大きい日数をカウント  
3. 残業時間＝勤務時間が8時間を超える分を合計  
4. 有給日数＝休暇区分が「有給」の件数をカウント  
5. それらを社員ごとに集計し、Excelに出力  

---

## 💻 使用技術
- **pandas**（条件抽出・groupby集計・concat結合）
- **openpyxl**（Excelファイル書き出し）
- **numpy**（条件演算・欠損補完など）
