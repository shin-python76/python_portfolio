# 💼 Python Freelance Portfolio

## 🧭 概要
このリポジトリは、Pythonを活用した業務自動化・データ処理・Webスクレイピングの  
**模擬案件（Mock Projects）** をまとめたポートフォリオです。  
クラウドソーシングなどでの応募時に、実務スキルの参考資料として公開しています。

---

## 📘 模擬案件一覧（予定）

| No | 案件名 | 使用技術 | 内容 |
|----|--------|----------|------|
| ① | 複数店舗の売上データ統合 | pandas / openpyxl / glob | 複数CSVを読み込み、店舗別集計表をExcel出力 |[mock_01_sales_merge](./mock_01_sales_merge) |
| ② | 勤怠データ自動集計 | pandas / openpyxl | 勤務時間・残業・有給日数を月次レポート化 |

| ⑤ | 可視化ダッシュボード（Streamlit） | pandas / Streamlit | 売上データをWebアプリで見える化 |

## 🧩 使用ライブラリ
- pandas（データ集計・整形）
- openpyxl（Excel操作）
- matplotlib（グラフ作成）
- requests / BeautifulSoup（Webデータ収集）
- Streamlit（データ可視化アプリ）
